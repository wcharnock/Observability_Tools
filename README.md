# Snake Game

A minimal classic Snake implementation.

## Run locally

```bash
npm install
npm run dev
```

Open `http://localhost:4173`.

## Test

```bash
npm test
```

## Manual verification checklist

- Arrow keys and WASD move the snake.
- Snake grows and score increments after eating food.
- Game ends on wall/self collision.
- Pause/Resume works with button and `Space`/`P`.
- Restart works with button and `Enter` after game over.
- Mobile on-screen directional buttons move the snake.
